# FeedVerse 客户演示手册（操作版）

---

## 演示前准备

1. 确认后端 `npm run dev`（端口 3001）和前端 `npm run dev`（端口 3000）均已运行
2. Chrome 安装 MetaMask，切换到 **BSC Testnet**，钱包有少量 tBNB
3. 打开 `http://localhost:3000`，确认页面加载完毕

---

## 演示注意事项

1. **网络要求**：确保本机 3000 和 3001 端口未被占用
2. **MetaMask 准备**：提前导入演示账户并切换到 BSC Testnet
3. **备用方案**：如果钱包连接失败，跳过演示十，其他功能均可正常展示（Mock 数据）
4. **屏幕分辨率**：建议 1920×1080 全屏展示效果最佳
5. **页面切换节奏**：每个模块停留 1-2 分钟，重点展示交互动画效果
